# Netflix Clone with React

## Introduction
Netflix, Inc. is an American over-the-top content platform streaming service and production company headquartered in Los Gatos, California. Netflix was founded in 1997 by Reed Hastings and Marc Randolph in Scotts Valley, California.

<br />

Together, we created a clone of one of the most popular content platform and streaming service. 

## Features
We have made the frontend of Netflix. In a developer perspective the frontend of this app is very complicated and 
responsive. This is a practice project for all wannabe frontend devs.
